### Spécifications Techniques

| **Spécifications**                                       | **Explication**                                                                                                                                                                    | **Compléments**                                     |
| -------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| STx1 : **Architecture**                                  | - Modèle-Vue-Controleur                                                                                                                                                            | séparé interface graphique et logique               |
| STx2 : **Conformité du code**                            | - bonne balisation<br>- validée par le validateur W3C.<br>- CSS bien structuré<br>- POO obligatoire dans Php<br>- respecter [PSR-12](https://www.php-fig.org/psr/psr-12/) pour Php | code clair, structuré et **commenté**               |
| STx3 : **Contrôle des champs de saisie des formulaires** | - valider en front et back end les saisies                                                                                                                                         |                                                     |
| STx4 : **Interdiction d’utiliser les CMS**               |                                                                                                                                                                                    |                                                     |
| STx5 : **Framework**                                     | - Framework interdit<br>- Préprocesseur CSS autorisé                                                                                                                               |                                                     |
| STx6 : **Stack technique**                               | - Serveur : Apache<br>- Frontend : HTML5 / CSS3 / JS<br>- Backend : **PHP**<br>- Base de données : SGBD SQL                                                                        | *exemple SGB*D : MySQL, PostgreSQL, MariaDB, …      |
| STx7 : **Moteur de template**                            | - Utiliser un moteur de template pour backend                                                                                                                                      | *ie.* notamment les inclusions de fragments de page |
| STx8 : **Utiliser des clés étrangères**                  | - pour BDD                                                                                                                                                                         |                                                     |
| STx9 : **Vhost**                                         | - utiliser Wamp pour vhost<br>- accueille le contenu statique                                                                                                                      | *exemple* : images, CSS, JS…                        |
| STx10 : **Responsive Design**                            | - s'adapter a la taille de l'écran<br>- menu burger pour mobile                                                                                                                    | *exemple* : mise en page, menus, texte, images…     |
| STx11 : **Sécurité**                                     | - info de connexion dans cookies sécurisés<br>- info sensible pas conservé en clair<br>- défense contre attaque<br>- https                                                         | *exemple* : injections SQL, XSS, CSRF               |
| STx12 : **SEO**                                          | - balise Meta<br>- html bien structuré avec balises<br>- chargement page < 3sec<br>- nom url cohérent<br>- fichier sitemap.xml et robots.txt                                       |                                                     |
| STx13 : **Routage d'URL**                                | - URLs lisibles, hiérarchisées et cohérentes<br>- routage côté Backend                                                                                                             |                                                     |
| STx14 : **Tests unitaires**                              | - faire pour au moins un contrôleur à l'aide de PHPUnit                                                                                                                            |                                                     |
